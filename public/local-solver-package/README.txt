# Local Scheduler Optimizer Package 
 
## Quick Start: 
1. Extract all files to a folder 
2. Run: start_local_solver.bat (Windows) or start_local_solver.sh (Mac/Linux) 
3. Keep the window open while using your webapp 
 
## Files included: 
- start_local_solver.bat/sh: One-click launcher 
- fastapi_solver_service.py: Advanced solver (RECOMMENDED) 
- local_solver.py: Basic fallback solver 
- scheduler_sat_core.py: Core optimization engine 
 
The advanced solver will automatically detect and use 
testcase_gui.py if you place it in the parent folder.
